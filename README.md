## Projects

**01 — scRNA-seq Analysis of the Pancreatic Cancer Tumor Microenvironment**
Single-cell RNA-seq analysis of PDAC tumor microenvironments using GEO dataset GSE155698. QC, clustering, canonical marker annotation, and per-patient differential expression in R using Seurat.
[View repo](https://github.com/sim-randhawa/scrnaseq-pdac-tme-analysis)

**02 — TCGA PAAD Survival Analysis**
End-to-end clinical survival analysis of 185 pancreatic adenocarcinoma patients using TCGA-PAAD data. Kaplan-Meier estimation, log-rank testing, and multivariate Cox regression in R.
[View repo](https://github.com/sim-randhawa/tcga-paad-survival-analysis)

**03 — CMS Medicare Provider Billing Analysis**
SQL-driven analysis of 223K+ CMS Medicare Part B billing records. Built a SQLite database and used multi-table joins, window functions, and subqueries to surface payment patterns across specialties.
[View repo](https://github.com/sim-randhawa/cms-medicare-sql-analysis)
